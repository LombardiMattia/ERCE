COS'È ERCE?

ERCE è una piattaforma di e-commerce educativa sviluppata da Lombardi Mattia, studente dell’Istituto Tecnico Tecnologico "S. Cannizzaro". Il sito nasce come progetto didattico per simulare un vero negozio online e mettere in pratica competenze in programmazione, design web e gestione dei dati.

Chi siamo

Lombardi Mattia

Studente appassionato di informatica, design e sviluppo web. Ha realizzato ERCE come progetto finale scolastico per dimostrare le proprie abilità tecniche e creative.

Obiettivi del Progetto

  • ✅ Simulare un sistema e-commerce completo
  • ✅ Utilizzare HTML, CSS, JS e JSON senza backend
  • ✅ Produrre fatture PDF lato client
  • ✅ Applicare principi di responsive design

Nota dello sviluppatore

ERCE funziona localmente tramite un file JSON statico. I prodotti visibili sono quelli presenti nel JSON, senza uso di Node.js o database esterni. Il sito è interamente realizzato con HTML, CSS, JavaScript e JSON.
